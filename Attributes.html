href attribute

HTML links are defined with a tag. The link address is specified in the href attribute.
Visit w3schools

src attribute

HTML images are defined with the img tag and the filename of the image source is specified.

alt attribute

Specifies an alternate text for an image , if the image for some reason cannot be displayed.

style attribute

Used to add styles to an element such as color,font and size.

lang attribute

Lang should be included inside the html tag to declare the language of the web page.

Title attribute

Defines some extra information about an element.
